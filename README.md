- Sistema-de-Irriga-o-Automatizado -

Projeto de um grupo de universitarios da 1 fase de analise e desenvolvimento de sistemas. 

desenvolvido com HTML e que apresenta uma solução de irrigação automatizada para jardins e plantas residenciais.

O objetivo do projeto é demonstrar como a utilização de sensores de umidade junto com um arduino pode auxiliar no cuidado das plantas, acionando automaticamente a irrigação quando o solo apresenta baixos níveis de umidade.

O site apresenta de forma visual e organizada o problema identificado, o público-alvo, a solução proposta, a proposta de valor e a implementação do sistema utilizando Arduino.

-> Tecnologias Utilizadas

* HTML5
* Arduino (conceito do projeto)

-> Objetivo: 

Desenvolver uma solução tecnológica que facilite os cuidados com plantas e jardins, promovendo praticidade, automação e uso eficiente da água.
